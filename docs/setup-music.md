# Set Music Config

## What does this block do?
This block helps you prepare the rules for your musical instrument. It decides the musical scale, the first note, and how many notes you can play in total!

## How does it work?
You place this block inside your `on start` loop. You can choose:
- **Scale root and type:** Like "C Major" or "A Minor", which sets the mood of the music.
- **Starting note and octave:** Like "G 3", which tells the micro:bit the lowest note it should start playing from.
- **Notes available:** A number like `20` that sets the limit on how high you can go. 

## Why would you use it?
Instead of using lots of different blocks to change how the music sounds, this single "helper block" makes it super easy to set up your instrument all at once. This saves time and helps keep your code neat!
